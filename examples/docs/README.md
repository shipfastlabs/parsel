# Example Documents

These files are used by the example scripts in this directory.

- `sample.pdf` is an Apple annual report used by the PDF example.
- `sample.docx` is Microsoft's `SampleDoc.docx` from the OfficeDev `Office-Add-in-samples` repository.
- `sample.xlsx` is Microsoft's `SalesTemplate.xlsx` from the OfficeDev `Office-Add-in-samples` repository.
- `sample.png` is a copy of the Parsel project image from `art/og.png`.
